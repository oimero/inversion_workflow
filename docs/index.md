# WTIE API 文档

这是 WTIE 库的内部 API 参考文档，目标是让同事可以快速定位模块、类和函数。

## 文档范围

- 第一版聚焦 API 检索，不包含完整教程
- 文档以中文为主
- 优先覆盖顶层公开入口与核心子包

## 快速开始

1. 安装文档依赖

```bash
pip install -r requirements-docs.txt
```

2. 启动本地预览

```bash
mkdocs serve
```

3. 打开浏览器访问

```text
http://127.0.0.1:8000
```

## 入口说明

建议从 API 总览页面进入，再按子包查看具体接口。
